## Project with leaflet

Starting with Docker

You can use either cmd or gitBash both works fine

```bash
docker-compose up
```

## for force build

```bash
docker-compose up --build
```

Use ctrl + c to stop the docker

## cd frontend

````bash
npm run dev```
````

## cd backend -- for development

````bash
npm run start:dev```

## cd backend -- for production

```bash
npm run start:prod```
````
